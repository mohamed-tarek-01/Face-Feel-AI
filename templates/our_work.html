Our Work

Facial Emotion Recognition with FER2013

At Face Feel AI, we developed a facial emotion recognition system using the FER2013 dataset. This dataset contains thousands of labeled images representing different emotions, providing a strong foundation for training our deep learning model.

Data Preprocessing & Augmentation

To enhance model performance, we applied data augmentation techniques, including:

  • Rotation to account for slight head tilts.
  • Flipping to introduce variations in facial orientation.
  • Brightness adjustments to improve robustness under different lighting conditions.
  • Zooming & Cropping to create diverse training samples.

Model Development - CNN

We built a CNN model specifically designed for facial emotion recognition. The architecture included:

  • Multiple convolutional layers to extract essential facial features.
  • Batch normalization & dropout layers to prevent overfitting.
  • Fully connected layers for final emotion classification.

After rigorous training and fine-tuning, our model achieved an accuracy of 68%, which is competitive for the FER2013 dataset.

Deployment with Flask

To make the model accessible, we deployed it using Flask, a lightweight web framework. The deployment involved:

  • Developing a user-friendly web interface for image uploads and real-time predictions.
  • Integrating the trained CNN model to process user-submitted images.
  • Hosting the Flask application for seamless interaction with users.
